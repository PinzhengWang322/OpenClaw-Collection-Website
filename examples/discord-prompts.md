# Sanitized Discord Prompt Examples

下面这些不是原始聊天记录，而是按真实使用习惯整理出来的公开版示例。

## Add A New Card

```text
新增电影 Begin Again
类型 音乐 / 剧情 / 电影
时间 25.09
产地 美国
主题 音乐，纽约，重新开始
```

## Update A Field

```text
把 Begin Again 的观看时间改成 25.10
```

## Replace A Cover

```text
把泰拉瑞亚灾厄的封面换成这张，然后把这几张也加到详情页。
```

## Delete A Card

```text
删掉那张测试用的黑镜条目，只保留白色圣诞节。
```

## Add A Travel Entry

```text
新增旅游卡片：云南旅行。
时间 25.08，事件旅行，城市云南，印象山景和慢节奏。
```

## Good Agent Response Style

```text
做好了。
```

或者稍微多一点点：

```text
做好了，时间已改成 25.10，排序也同步更新了。
```

这类维护任务里，短确认通常比长解释更自然。
